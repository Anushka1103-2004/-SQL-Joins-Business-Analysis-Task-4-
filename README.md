# SQL Joins – Business Analysis (Task 4)

## Objective
To analyze business data using SQL joins (INNER JOIN, LEFT JOIN) and answer real-world business questions.

## Tools Used
- SQLite (DB Browser)
- Chinook Database

## Tasks Completed
- Loaded customers, orders, products, categories tables
- Used INNER JOIN and LEFT JOIN
- Identified customers with no orders
- Calculated revenue per product
- Generated category-wise revenue
- Exported joined output to CSV
- Derived business insights

## Files Included
- joins_queries.sql → All SQL queries with comments
- joined_output.csv → Final joined output
- insights.txt → Business insights

## Key Learning
- How joins combine data from multiple tables
- Business usage of LEFT JOIN
- Revenue and customer analysis using SQL
